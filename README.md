# leseecke

**leseecke** soll als dein persönliches Archiv dienen. Speichere und bewerte die Bücher oder halte einfach deine Gedanken direkt fest.

### Features

- **📖 Bücher speichern:** Verwalte Titel, Autor, Jahr und Genre in einer zentralen Übersicht.
- **⭐ Bewertungen:** Gib jedem Werk eine Stern-Bewertung.
- **📝 Notizen machen:** Hinterlege persönliche Gedanken, Rezensionen oder Lieblingszitate direkt zu den Büchern.

Perfekt für jede Person, die gerne ihre gelesenen Bücher an einem Platz sammeln möchte.
